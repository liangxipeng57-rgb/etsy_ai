# Etsy AI Studio

一个面向 Etsy 与跨境电商卖家的 AI 内容生产工作台，围绕商品上架、视觉素材制作、SEO 优化和短视频生成，提供一站式工作流。

## 项目简介

`Etsy AI Studio` 的目标是把电商卖家最常见的内容生产任务收敛到一个界面里完成，包括商品场景图生成、细节图增强、尺寸标注、宣传图合成、SEO 文案生成以及商品展示视频生成。项目采用 React + Vite 构建，前端 UI 按工作台形态重构，并把 API 配置入口统一放到了设置页中。

## 功能模块

### 1. 主生成器

- 上传商品图后，可同时生成商品场景图与 SEO 文案。
- 支持选择风格、图片尺寸、输出数量和商品尺寸信息。
- 适合新品快速上架和单品批量包装。

### 2. 图片工具

- 模特场景：生成更贴近穿搭或使用场景的商品展示图。
- 细节增强：突出面料、做工、纹理和局部卖点。
- 尺寸标注：自动生成白底尺寸说明图。
- 多图融合：把多张素材合并成一个完整画面。
- 宣传图生成：支持图生图与文生图两种方式，快速产出营销素材。

### 3. 视频工具

- 基于商品图生成动态展示视频。
- 支持自定义镜头提示词与视频时长。
- 适合详情页展示、社媒传播和广告投放素材制作。

### 4. SEO 工具

- 支持关键词生成 SEO 内容。
- 支持通过 Etsy 商品链接分析并生成优化文案。
- 输出标题、描述、标签，便于商品上架和搜索曝光优化。

### 5. 设置中心

- 在设置页统一录入和校验 API Key。
- 当前所有 AI 能力共用同一个 API 入口。
- 配置结果保存在浏览器本地，方便本地演示和团队内部部署。

## 应用场景

- Etsy 新品上架：快速生成主图场景、详情辅助图和 SEO 文案。
- 老品重做：重新包装视觉素材和搜索文案，提升点击率与转化率。
- 社媒营销：输出适合 Instagram、Pinterest、TikTok 等渠道的宣传图和短视频。
- 多平台分发：同一商品快速生成不同尺寸、不同风格、不同用途的素材。
- 团队协作：运营、设计、选品可围绕同一工作台完成内容生产与整理。

## 快速开始

### 环境要求

- Node.js 18 及以上
- npm 9 及以上

### 安装依赖

```bash
npm install
```

### 启动开发环境

```bash
npm run dev
```

默认访问地址：

```text
http://localhost:3000
```

### 首次使用

1. 打开应用后进入“设置”页面。
2. 在 `Unified API Key` 输入框中填写可用的 API Key。
3. 点击“验证并启用”。
4. 验证成功后即可使用图片、SEO、视频等功能。

## 配置说明

### API 配置

- 当前项目使用统一 API Key，不再区分前端多个 provider 配置入口。
- API Key 存储在浏览器 `localStorage` 中，适合本地使用和演示环境。
- 模型映射在 `services/providerRegistry.ts` 中维护，前端设置页只负责录入和校验密钥。

当前内置模型如下：

- 文本模型：`gpt-5.2`
- 图像编辑模型：`gemini-3-pro-image-edit`
- 文生图模型：`gemini-3-pro`
- 视频模型：`kling-v2.6-pro-i2v`

### 本地代理

开发环境下，Vite 已在 `vite.config.ts` 中配置代理转发。若你的网络环境需要代理访问外部接口，可以采用以下两种方式：

- 启动前设置系统环境变量，如 `HTTPS_PROXY`
- 修改 `vite.config.ts` 中的 `DEV_FALLBACK_PROXY`

如果你的机器本身可以直接访问目标接口，则无需额外修改。

## 典型使用流程

### 主生成器工作流

1. 上传 1 到 3 张商品图。
2. 选择风格、尺寸和输出数量。
3. 按需填写商品长宽高。
4. 勾选生成图片、生成 SEO 文案。
5. 等待结果输出并下载素材。

### 宣传图工作流

1. 选择图生图或文生图模式。
2. 上传背景图与商品图，或输入宣传图描述。
3. 选择尺寸和生成数量。
4. 批量预览并下载结果。

### 视频工作流

1. 上传商品图。
2. 输入镜头提示词。
3. 选择视频时长。
4. 生成并下载 MP4 视频。

## 项目结构

```text
.
├─ components/         # 页面模块与通用 UI 组件
├─ services/           # AI 服务封装、Provider、Prompt 与注册表
├─ utils/              # 文件处理等工具函数
├─ App.tsx             # 应用主入口与工作台布局
├─ index.tsx           # React 挂载入口
├─ index.html          # 页面模板与设计 token
├─ types.ts            # 全局类型定义
└─ vite.config.ts      # Vite 配置与本地代理
```
